Aprendo direttamente questo progetto cliccando il fiile "index.html" potresti non vedere correttamente la mappa o il tracciato GPX.
Questo problema è causato dalle politiche CORS (Cross-Origin Resource Sharing) del browser. Quando apri direttamente il file HTML nel browser tramite il file system locale (file:///), il browser considera l'origine (origin) come "null" anziché un protocollo HTTP o HTTPS valido. Di conseguenza, le richieste cross-origin vengono bloccate per motivi di sicurezza.
